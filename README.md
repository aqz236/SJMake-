## -非常荣幸我蹩脚的代码可以成为大家的轮子嘿嘿
本程序的功能是批量、随机生成指定地区，指定运营商的手机号码，并进行地级市遍历进行号码归属地分类


参数说明:
phoneHead.seven.randomSeven('A','B')

A:为省份名称
B:为运营商名称

例子：
phoneHead.seven.randomSeven('广西','移动')
